Supabase Setup Assistant - Complete in 20 Minutes

1 Run Database Migration
Creates all 12 database tables (profiles, menu_items, events, orders, etc.)
Time: 3 minutes
Instructions:
1. Click "Open SQL Editor" button above
2. Click "Copy Migration SQL" button
3. Paste into the SQL Editor
4. Click RUN button
5. Wait for "Success" message
6. Click "Mark Complete" above
2 Load Test Data (Optional)
Adds sample menu items, events, and products for testing
Time: 2 minutes
3 Configure Authentication URLs
Set allowed redirect URLs for authentication
Time: 2 minutes
Site URL: http://localhost:3000
Redirect URLs: http://localhost:3000/**
Instructions:
1. Click "Open Auth Settings"
2. Enter Site URL: http://localhost:3000
3. Enter Redirect URLs: http://localhost:3000/**
4. Scroll down and click SAVE
4 Configure Email Templates
Set up 5 branded email templates for authentication
Time: 15 minutes
For each template:
1. Click "Open Email Templates" above
2. Click "Open Template Guide" to open SUPABASE_EMAIL_TEMPLATES.md
3. For each of the 5 templates:
- Click template name in Supabase
- Copy Subject and Body from guide
- Paste into Supabase
- Click Save
